# Rooted: Gamified Mental Health Tracker

## Overview
Rooted is a web-based application that was developed as a final project to Harvard's CS50. It transforms the classic, clinicial mental health tracker into a gamified experience that encourages good wellness habits, while also feeling effortless.

## Key features:
* **Secure User Authentication:** Has password hashing and secure session management
* **Database Integration:** Utlizies SQL databases to store and query user metrics dynamically.
* **Dynamic Interface:** Built using Python (Flask) for a smooth experience.

## What I Learned:
Through developing Rooted, I gained hands-on experience with:
* Structuring a full stack application from scratch
* Debugging backend logic
* Managing data structures and handling user inputs safely
